# Todo List App

A simple and efficient web-based Todo List application that helps users manage their daily tasks effectively.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks by status (all, completed, pending)
- Responsive design for seamless use on different devices
- Persistent data storage (localStorage)

## Tech Stack

- **Frontend:** React, Bootstrap

## Installation

### Prerequisites
- Node.js installed

### Steps

1. Clone the repository:
   ```sh
   git clone https://github.com/omarmechergui/todo-list-app-web.git
   ```
2. Navigate to the project folder:
   ```sh
   cd todo-list-app-web
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```

## Usage

- Open the app in your browser at `http://localhost:3000`
- Add tasks using the input field and manage them as needed
